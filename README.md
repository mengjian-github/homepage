## 更新原则

这个站点的文案更新遵循三条原则：

1. **统一口径**：与 `USER.md`、GitHub 个人主页保持一致
2. **少写易过期数字**：优先写长期有效的人设与成果类型
3. **先承接品牌，再承接细节**：先让访问者知道“你是谁、做什么、为什么值得关注”
